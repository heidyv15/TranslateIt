# TranslateIt
Project 6 - TranslateMe

Submitted by: Heidy Veliz

TranslateMe is a translation app that allows users to enter a word, phrase, or sentence and translate it from English to Spanish using the MyMemory API. The app also keeps a history of translations and allows users to clear that history.

Time spent: X hours spent in total

Required Features

The following required functionality is completed:

 Users can enter a word, phrase, or sentence into a text field
 Users can tap a button to translate the input text
 The translated text appears in a separate output field
 The app translates from one language to another (English → Spanish)
 A history of translations is displayed in a list
 Users can clear the translation history
Stretch Features

The following stretch features are implemented:

 Multiple language selection
 Enhanced UI design / styling

Notes

One challenge encountered while building the app was setting up Firebase for data persistence. Due to setup complexity and time constraints, a local in-memory solution was used to store translation history instead. Another challenge was resolving Xcode build errors related to package dependencies and module imports.
